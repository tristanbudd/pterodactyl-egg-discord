## Installation

To install this egg :

1. Go into Admin panel
2. Go to "Nests"
3. Click "Import Egg"
4. Choose the .json file named "egg-discordjs-14"
5. And import !

## Usage

To use it, just create a server with this egg in the "Nest Configuration" section.
